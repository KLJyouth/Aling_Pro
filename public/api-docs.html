身份认证

API密钥认证

所有API请求都需要在请求头中包含您的API密钥。您可以在 用户控制台 中获取API密钥。 

curl -X POST "https://api.alingai.com/v1/chat/completions" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "messages": [
      {"role": "user", "content": "Hello, AlingAI!"}
    ]
  }'
请求头 说明
Authorization Bearer token，格式为 Bearer YOUR_API_KEY
Content-Type 必须设置为 application/json

安全提醒

请妥善保管您的API密钥，不要在客户端代码中暴露。 如果密钥泄露，请立即在控制台中重新生成。

对话接口

POST /v1/chat/completions

创建一个新的对话完成请求。支持多轮对话和流式响应。

请求参数

参数 类型 必需 说明
messages array 是 对话消息数组
model string 否 使用的模型，默认为 gpt-4
temperature number 否 温度参数，范围 0-2，默认 1
max_tokens integer 否 最大输出token数，默认 4096
stream boolean 否 是否启用流式响应，默认 false

请求示例

{
  "messages": [
    {
      "role": "system",
      "content": "你是一个有用的AI助手。"
    },
    {
      "role": "user",
      "content": "请解释一下什么是量子计算。"
    }
  ],
  "model": "gpt-4",
  "temperature": 0.7,
  "max_tokens": 2048,
  "stream": false
}

响应示例

成功响应 (200 OK)
{
  "id": "chatcmpl-abc123",
  "object": "chat.completion",
  "created": 1677652288,

    {
      "index": 0,

Python SDK

安装

pip install alingai

快速开始

import alingai

# 初始化客户端
client = alingai.Client(api_key="your-api-key")

# 创建对话
response = client.chat.completions.create(
    messages=[
        {"role": "user", "content": "Hello, AlingAI!"}
    ],
    model="gpt-4"
)

print(response.choices[0].message.content)

流式响应

stream = client.chat.completions.create(
    messages=[
        {"role": "user", "content": "写一首关于AI的诗"}
    ],
    model="gpt-4",
    stream=True
)

for chunk in stream:
    if chunk.choices[0].delta.content is not None:
        print(chunk.choices[0].delta.content, end="")
