🚀 快速开始

环境要求

  • PHP 8.1+ (推荐 8.2)
  • MySQL 8.0+ 或 SQLite 3.0+
  • Nginx 1.20+ 或 Apache 2.4+
  • Redis 6.0+ (可选，用于缓存)
  • Node.js 18+ (前端构建)

1. 克隆项目

git clone https://github.com/your-org/alingai-pro.git cd alingai-pro

2. 安装依赖

# PHP依赖 composer install --no-dev --optimize-autoloader # 前端依赖 npm install

3. 配置环境

# 复制环境配置文件 cp .env.example .env # 编辑配置文件 nano .env

4. 初始化系统

# 运行一键安装脚本 php install/install.php # 或者使用安装向导 ./install.sh

5. 启动服务

# 开发环境 ./start.sh # 生产环境 ./deploy.sh
提示： 首次启动后，请访问 http://localhost:8000/admin.html 进行系统初始化配置。
