代码示例

简单易用的API接口

Python示例

# 安装SDK pip install alingai-sdk # 使用示例 from alingai import QuantumSecurity # 初始化客户端 client = QuantumSecurity(api_key="your_api_key") # 生成量子密钥 key = client.generate_quantum_key() # 加密数据 encrypted_data = client.encrypt(data, key) # 解密数据 decrypted_data = client.decrypt(encrypted_data, key)

JavaScript示例

// 安装SDK npm install @alingai/sdk // 使用示例 import { QuantumSecurity } from '@alingai/sdk'; // 初始化客户端 const client = new QuantumSecurity({ apiKey: 'your_api_key' }); // 异步加密操作 const encrypted = await client.encrypt(data); // 异步解密操作 const decrypted = await client.decrypt(encrypted);
