开始使用

AlingAi API 提供一系列强大的人工智能功能，包括对话生成、图像创建和语音合成等。本文档将帮助您快速了解如何使用这些API。

基本要求
  • 您需要有一个有效的 AlingAi 账户
  • 获取 API 密钥（在API密钥页面创建）
  • 确保账户中有足够的使用额度
API基础地址
API基础地址 
https://api.alingai.pro/v1

认证方式

所有API请求都需要使用API密钥进行认证。您可以在请求头中添加您的API密钥。

HTTP请求头 
Authorization: Bearer YOUR_API_KEY
安全提示： 请妥善保管您的API密钥，不要在客户端代码中暴露它。建议通过服务器中转API请求。

对话API

对话API允许您与AlingAi的对话模型进行交互，生成自然、连贯的对话响应。

请求格式
参数 类型 描述
model必填 string 要使用的模型 ID，例如 "alingai-chat-1"
messages必填 array 对话消息数组，每条消息包含 role 和 content
temperature number 控制随机性，0-1之间，默认为0.7
max_tokens integer 生成文本的最大长度，默认为2048
cURL
Python
JavaScript
PHP
cURL 示例 
curl -X POST https://api.alingai.pro/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "model": "alingai-chat-1",
    "messages": [
      {"role": "system", "content": "你是一个有用的智能助手。"},
      {"role": "user", "content": "给我介绍一下人工智能的发展历程。"}
    ],
    "temperature": 0.7,
    "max_tokens": 2048
  }'
Python 示例 
import requests
import json

url = "https://api.alingai.pro/v1/chat/completions"
headers = {
    "Content-Type": "application/json",
    "Authorization": "Bearer YOUR_API_KEY"
}
data = {
    "model": "alingai-chat-1",
    "messages": [
        {"role": "system", "content": "你是一个有用的智能助手。"},
        {"role": "user", "content": "给我介绍一下人工智能的发展历程。"}
    ],
    "temperature": 0.7,
    "max_tokens": 2048
}

response = requests.post(url, headers=headers, data=json.dumps(data))
print(response.json())
JavaScript 示例 
fetch('https://api.alingai.pro/v1/chat/completions', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': 'Bearer YOUR_API_KEY'
  },
  body: JSON.stringify({
    model: 'alingai-chat-1',
    messages: [
      {role: 'system', content: '你是一个有用的智能助手。'},
      {role: 'user', content: '给我介绍一下人工智能的发展历程。'}
    ],
    temperature: 0.7,
    max_tokens: 2048
  })
})
.then(response => response.json())
.then(data => console.log(data))
.catch(error => console.error('Error:', error));
PHP 示例 
$url = 'https://api.alingai.pro/v1/chat/completions';
$data = array(
    'model' => 'alingai-chat-1',
    'messages' => array(
        array('role' => 'system', 'content' => '你是一个有用的智能助手。'),
        array('role' => 'user', 'content' => '给我介绍一下人工智能的发展历程。')
    ),
    'temperature' => 0.7,
    'max_tokens' => 2048
);

$options = array(
    'http' => array(
        'header'  => "Content-type: application/json\r\nAuthorization: Bearer YOUR_API_KEY\r\n",
        'method'  => 'POST',
        'content' => json_encode($data)
    )
);

$context  = stream_context_create($options);
$result = file_get_contents($url, false, $context);

if ($result === FALSE) { 
    echo "Error"; 
} else {
    var_dump(json_decode($result));
}

图像生成API

图像生成API允许您根据文本描述生成高质量的图像。

请求格式
参数 类型 描述
prompt必填 string 图像描述文本
size string 生成图像尺寸，可选值："512x512", "1024x1024"，默认为"512x512"
n integer 生成图像数量，默认为1
图像生成示例 
curl -X POST https://api.alingai.pro/v1/images/generations \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "prompt": "一只可爱的小猫坐在窗台上看日落",
    "size": "1024x1024",
    "n": 1
  }'

语音合成API

语音合成API可以将文本转换为自然、流畅的语音。

请求格式
参数 类型 描述
text必填 string 要转换为语音的文本
voice_id string 语音ID，可选值："female1", "male1", "female2", "male2"，默认为"female1"
format string 输出格式，可选值："mp3", "wav"，默认为"mp3"
语音合成示例 
curl -X POST https://api.alingai.pro/v1/speech/synthesize \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "text": "欢迎使用AlingAi语音合成服务，我们提供自然流畅的语音效果。",
    "voice_id": "female1",
    "format": "mp3"
  }'

速率限制

为了确保服务质量并防止滥用，API请求受到速率限制。限制根据您的账户类型而有所不同。

账户类型 请求限制 每分钟请求数
基础版 20 req/min 20
高级版 60 req/min 60
企业版 200 req/min 200

错误处理

API使用标准HTTP状态码表示请求状态，并在响应体中提供详细的错误信息。

状态码 描述
200 请求成功
400 无效请求 - 请求参数错误
401 未授权 - API密钥无效或缺失
403 禁止访问 - 没有权限执行操作
429 请求过多 - 超出速率限制
500 服务器错误 - 内部服务错误
错误响应示例 
{
  "error": {
    "code": "invalid_request_error",
    "message": "必须提供prompt参数",
    "param": "prompt",
    "type": "invalid_parameter"
  }
}

最佳实践

以下是使用AlingAi API的一些最佳实践建议：

安全性
  • 不要在客户端代码中暴露API密钥
  • 使用后端服务中转API请求
  • 定期轮换API密钥
性能优化
  • 缓存常用响应以减少API调用
  • 对于长对话，考虑使用摘要技术减少历史消息长度
  • 优化prompt设计，提供清晰、具体的指示
成本控制
  • 合理设置max_tokens参数，避免生成过长无用内容
  • 根据需求选择适当的模型，不同模型有不同的定价
  • 实现请求限制和监控，避免意外超额使用
