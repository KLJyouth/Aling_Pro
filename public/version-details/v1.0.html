v1.0 安装部署

简单快捷的部署流程

系统要求

硬件要求

• CPU: 双核 2.0GHz 或以上
• 内存: 4GB RAM 最小，8GB 推荐
• 存储: 50GB 可用空间
• 网络: 100Mbps 网络连接

软件要求

• 操作系统: Ubuntu 18.04+ / CentOS 7+
• Web服务器: Apache 2.4+ / Nginx 1.14+
• 数据库: MySQL 5.7+ / MariaDB 10.2+
• PHP: 7.2+ (推荐 7.4)

安装步骤

1
下载安装包
从官网下载v1.0安装包
2
环境准备
安装PHP、MySQL等依赖
3
数据库配置
创建数据库并导入结构
4
系统初始化
运行安装向导完成配置

快速安装命令 

# 1. 下载安装包
wget https://releases.alingai.com/v1.0/alingai-platform-v1.0.tar.gz

# 2. 解压到web目录
tar -xzf alingai-platform-v1.0.tar.gz -C /var/www/html/

# 3. 创建数据库
mysql -u root -p -e "CREATE DATABASE alingai_v1;"

# 4. 导入数据库结构
mysql -u root -p alingai_v1 < /var/www/html/alingai/database/schema.sql

# 5. 设置文件权限
chmod -R 755 /var/www/html/alingai/
chown -R www-data:www-data /var/www/html/alingai/

# 6. 访问安装向导
# 浏览器打开: http://your-server/alingai/install/
