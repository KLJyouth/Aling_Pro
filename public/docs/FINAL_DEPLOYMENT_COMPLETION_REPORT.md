## 🎯 系统架构

### 前端技术栈
- **核心框架**: HTML5 + CSS3 + JavaScript ES6+
- **UI框架**: Tailwind CSS + Custom CSS
- **动画引擎**: GSAP + Three.js
- **字体系统**: JetBrains Mono + Inter + Fira Code
- **图标库**: Font Awesome 6.7.2

### 后端技术栈
- **服务器语言**: PHP 8.1+
- **框架**: Slim Framework + PSR-7
- **数据库**: SQLite + 文件系统存储
- **API架构**: RESTful API
- **认证系统**: JWT Token

### 开发工具
- **包管理**: Composer
- **代码质量**: PHPStan + PHP CS Fixer
- **测试框架**: PHPUnit
- **容器化**: Docker (可选)
