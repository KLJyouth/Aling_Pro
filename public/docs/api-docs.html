身份认证

API密钥认证

所有API请求都需要在HTTP头中包含有效的API密钥： 

curl -H "Authorization: Bearer YOUR_API_KEY" \
     -H "Content-Type: application/json" \
     https://api.alingai.com/v1/user/profile
const response = await fetch('https://api.alingai.com/v1/user/profile', {
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  }
});

const data = await response.json();
import requests

headers = {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
}

response = requests.get('https://api.alingai.com/v1/user/profile', headers=headers)
data = response.json()
量子认证增强

启用量子认证后，每个请求都会使用量子密钥进行额外加密，提供量子级别的安全保护。

量子加密API

POST /quantum/encrypt 需要认证

使用量子加密算法对数据进行加密处理，支持多种量子加密协议。

请求参数
参数名 类型 必需 说明
data string 是 需要加密的数据
algorithm string 否 加密算法 (默认: "qkd-aes256")
key_id string 否 指定量子密钥ID
响应示例
{
  "success": true,
  "data": {
    "encrypted_data": "QkQ3N0Y4...", 
    "key_id": "qk_1234567890",
    "algorithm": "qkd-aes256",
    "quantum_signature": "qs_abcdef123456",
    "expiry": "2024-12-31T23:59:59Z"
  },
  "metadata": {
    "request_id": "req_xyz789",
    "processing_time": "45ms",
    "quantum_entropy": 99.87
  }
}
POST /quantum/decrypt 需要认证

解密量子加密的数据，需要提供对应的量子密钥。 

{
  "encrypted_data": "QkQ3N0Y4...",
  "key_id": "qk_1234567890",
  "quantum_signature": "qs_abcdef123456"
}

WebSocket实时通信

连接端点

建立WebSocket连接用于实时数据推送和事件通知。

WebSocket URL wss://api.alingai.com/v1/ws
认证方式 Query参数或首次消息
协议 JSON
连接示例
const ws = new WebSocket('wss://api.alingai.com/v1/ws?token=YOUR_API_KEY');

ws.onopen = function() {
  console.log('连接已建立');
  
  // 订阅威胁告警
  ws.send(JSON.stringify({
    type: 'subscribe',
    channel: 'threat_alerts',
    filters: {
      severity: ['high', 'critical']
    }
  }));
};

ws.onmessage = function(event) {
  const data = JSON.parse(event.data);
  console.log('收到消息:', data);
};

速率限制

API调用限制

为确保服务稳定性，我们对API调用频率进行了限制：

  • 免费版：每分钟100次请求
  • 专业版：每分钟1000次请求
  • 企业版：每分钟10000次请求
  • 量子加密API：每分钟50次请求（所有版本）

速率限制响应头

响应头 说明
X-RateLimit-Limit 速率限制值
X-RateLimit-Remaining 剩余请求次数
X-RateLimit-Reset 重置时间戳
